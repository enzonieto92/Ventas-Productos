# 🛒 Sistema de Punto de Venta

> Aplicación de escritorio para gestión de caja y ventas, pensada para negocios reales con equipos de bajo rendimiento.

---

## 📋 Descripción

Este proyecto es un sistema de punto de venta (POS) de escritorio, desarrollado con el objetivo de brindar una interfaz **simple e intuitiva** para el día a día de un negocio.

La decisión de desarrollarlo como aplicación de escritorio responde a una necesidad concreta: **compatibilidad con equipos de bajo rendimiento y sistemas operativos en desuso**, como Windows 7 de 32 bits, que siguen siendo comunes en entornos comerciales.

---

## ⚙️ Stack Tecnológico

| Herramienta | Versión / Detalle |
|---|---|
| Framework | .NET Framework 4.7.2 |
| IDE | Visual Studio 2022 Community |
| Base de datos | SQLite |
| SO compatible | Windows 7 (32 bits) o superior |

> **¿Por qué .NET Framework 4.7.2?**
> Si bien es un framework legado, fue la elección correcta para garantizar compatibilidad con Windows 7 de 32 bits, sistema operativo que sigue siendo utilizado en muchos negocios pequeños.

## 🚀 Instalación y Requisitos

### Requisitos del sistema

- Windows 7 SP1 (32 bits) o superior
- [.NET Framework 4.7.2](https://dotnet.microsoft.com/en-us/download/dotnet-framework/net472) instalado

### Pasos de instalación

1. Verificá que tenés instalado .NET Framework 4.7.2.
   - Si no lo tenés, descargalo desde el [sitio oficial de Microsoft](https://dotnet.microsoft.com/en-us/download/dotnet-framework/net472).
2. Descargá el instalador desde la rama principal [Descargar instalador](../../raw/master/Gestión%2Ventas.msi)
3. Ejecutá el instalador y seguí los pasos.
4. Una vez instalado, abrí la aplicación desde el acceso directo en el escritorio.

### Compilar desde el código fuente

```bash
# Clonar el repositorio
git clone https://github.com/enzonieto92/Ventas-Productos.git

# Abrir la solución en Visual Studio 2022
# Compilar con Ctrl+Shift+B o desde el menú Build > Build Solution
```
